# Сайт-визитка на Django

Современный сайт-визитка, созданный с использованием Django. Проект готов к расширению с Wagtail CMS.

## Структура проекта

- **core** - основное приложение с настройками Django
- **main** - приложение с простыми HTML страницами (главная, о нас, контакты)
- **portfolio** - приложение для управления портфолио проектов

## Возможности

- ✅ Современный адаптивный дизайн с анимациями
- ✅ Управление портфолио через админку Django
- ✅ Красивые HTML шаблоны с эффектами
- ✅ Готовность к интеграции с Wagtail CMS
- ✅ Админка для добавления проектов в портфолио
- ✅ Детальные страницы проектов
- ✅ Интерактивные эффекты и анимации
- ✅ Мобильная адаптивность

## Установка и запуск

### 1. Клонирование и установка зависимостей

```bash
# Создайте виртуальное окружение
python -m venv venv

# Активируйте виртуальное окружение
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# Установите зависимости
pip install -r requirements.txt
```

### 2. Настройка базы данных

```bash
# Создайте миграции
python manage.py makemigrations

# Примените миграции
python manage.py migrate

# Создайте суперпользователя
python manage.py createsuperuser
```

### 3. Запуск сервера

```bash
python manage.py runserver
```

Сайт будет доступен по адресу: http://127.0.0.1:8000/

## Админка

- Django админка: http://127.0.0.1:8000/admin/

## Структура страниц

### Основные страницы (main)
- `/` - Главная страница с современным дизайном
- `/about/` - О студии с информацией о команде
- `/contact/` - Контакты с формой обратной связи

### Портфолио (portfolio)
- `/portfolio/` - Список всех проектов с современным дизайном
- `/portfolio/<slug>/` - Детальная страница проекта

## Добавление проектов в портфолио

1. Зайдите в Django админку: http://127.0.0.1:8000/admin/
2. Войдите под учетными данными суперпользователя
3. Перейдите в раздел "Portfolio projects"
4. Нажмите "Add Portfolio project"
5. Заполните поля:
   - **Title** - название проекта
   - **Slug** - URL-адрес (автозаполняется)
   - **Description** - описание проекта
   - **Technologies** - используемые технологии
   - **Project URL** - ссылка на проект (опционально)
   - **GitHub URL** - ссылка на GitHub (опционально)
   - **Image** - изображение проекта (опционально)

## Технологии

- **Backend**: Django 4.2 (готов к интеграции с Wagtail)
- **Frontend**: Bootstrap 5, Font Awesome
- **Database**: SQLite (по умолчанию)
- **Static files**: WhiteNoise

## Настройка для продакшена

1. Создайте файл `.env` на основе `.env.example`
2. Измените `DEBUG=False` в настройках
3. Настройте базу данных PostgreSQL
4. Настройте статические файлы для продакшена
